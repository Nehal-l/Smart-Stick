# VISION-I:A BLIND ASSISTANCE SYSTEM
VISION-I is an intelligent vision enhancement module designed to assist visually impaired individuals in perceiving their surroundings safely and independently. The system combines real-time object detection, distance estimation, and voice-based feedback to help users identify obstacles and navigate effectively.

## Features
* Real-Time Object Detection
* Voice-Based Feedback
* Wireless Communication
* Lightweight & Easy to Integrate
* Obstacle Detection Mode 

## Setup
* Set Up the Project Environment
* Install Required Dependencies 
* Set Up the Model Files
* Configure Camera Source
* Run [MainFile](python webcam_blind_voice.py)

## Technical Stack
* Language: Python
* Libraries: TensorFlow, OpenCV, NumPy, PyTesseract, PyAudio, pyttsx3
* Model: SSD_MobileNet (Single Shot MultiBox Detector)
* Functionality: Object Detection + Distance Estimation + Voice Alerts

## Vision
The aim of VISION-I is to empower visually impaired individuals by combining AI and accessibility.By offering real-time perception, obstacle detection, and audio feedback, this system helps bridge the gap between human intuition and technological assistance.
